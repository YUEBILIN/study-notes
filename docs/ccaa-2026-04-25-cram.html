🎯二、A 档必背内容

A1管理评审（最高概率 ★★★★★）

（1）目的

最高管理者按策划的时间间隔对组织的管理体系进行评审，以确保其持续适宜性、充分性、有效性，并与组织战略方向保持一致。

（2）输入（9 项）

  1. 以往管理评审所采取措施的情况
  2. 与管理体系相关的内外部因素的变化（含战略方向变化）
  3. 管理体系绩效和有效性信息：
    • ① 顾客满意度和相关方反馈
    • ② 质量/环境/职业健康目标的实现程度
    • ③ 过程绩效、产品和服务的符合性
    • ④ 不符合和纠正措施
    • ⑤ 监视和测量结果
    • ⑥ 审核结果（内审+外审）
    • ⑦ 外部供方的绩效
  4. 资源的充分性
  5. 应对风险和机遇所采取措施的有效性
  6. 改进的机会

（3）输出（3 项）

  1. 改进的机会
  2. 管理体系变更的需求
  3. 资源需求
💡 记忆口诀："以内外，绩资险改" → 以前措施 / 内外部变化 / 绩效信息 / 资源 / 风险机遇 / 改进机会

A2多场所抽样规则（综合题必考计算）

核心公式（IAF MD 1 / CNAS-CC01）

审核类型抽样公式备注
初次认证y = √x（向上取整）x = 场所数
监督审核y = 0.6×√x（向上取整）
再认证审核y = 0.8×√x（向上取整）

必守规则

  1. 总部必审（每次都审，不参与抽样）
  2. 每次抽样要包含新场所（不能重复抽同一批）
  3. 不同地理位置/国家的场所要覆盖
  4. 有不符合的场所下次必须再审
  5. 生产/施工等独立运营场所 → 通常不适用抽样简化，应全部审核
  6. 活动不同类的场所 → 分类分别抽样

例题（XX 电气公司场景）

  • 总部苏州：必审 → 1
  • 2 个生产基地（苏州+无锡）：独立生产运营，不简化，全审 → 2
  • 3 个分公司（销售+安装）：√3 = 1.73 → 抽 2
  • 合计最少 5 个场所

A3不符合项判定 + 报告要素

不符合分类

类型判定原则
严重不符合① 体系某要素整体失效；② 区域性/系统性缺失；③ 对产品/服务符合性产生严重影响；④ 可能导致管理体系失效或法律法规违规
一般不符合（轻微）孤立、偶发、影响有限，未导致体系失效
观察项虽符合但存在改进空间，或证据不足以判为不符合

不符合报告 9 要素（5W1H 展开）

  1. 受审核方名称 + 受审核部门
  2. 审核日期、审核员姓名
  3. 不符合事实描述（5W1H）：What 发现什么 / Where 哪里发现 / When 何时发现 / Who 涉及谁 / Which 哪份文件 / How 如何不符合
  4. 违反的准则条款（如 GB/T 19001-2016 第 8.3.6 条）
  5. 不符合类型（严重/一般）
  6. 审核组长签字
  7. 受审核方确认签字
  8. 纠正 + 纠正措施完成期限
  9. 验证方式与完成日期
事实描述模板：
"2026 年 4 月 26 日，审核组在 XX 部门审核时发现：[5W1H 事实]。此做法不符合 GB/T 19001-2016《质量管理体系 要求》第 X.X.X 条 '[原文摘录]' 的规定。"

A4审核结论形成流程

审核组内部会议主要工作（7 项）

  1. 审核员各自汇报审核发现
  2. 汇总所有审核证据，确认与审核准则一致
  3. 评审审核发现，判定符合/不符合/观察项
  4. 形成不符合项报告，审核组长签字
  5. 讨论审核目标是否达成、审核范围是否覆盖
  6. 形成审核结论（是否推荐认证）
  7. 准备末次会议陈述内容

审核结论应考虑的因素

  • 管理体系符合审核准则的程度
  • 管理体系有效实施和保持的情况
  • 内审和管理评审的有效性
  • 审核目标达成情况
  • 识别出的风险与改进领域

推荐结论的 3 类

  1. 推荐通过认证/注册
  2. 有条件推荐（需整改验证后）
  3. 不予推荐（严重不符合/系统性失效）

📎三、B 档保底内容

B1. 审核范围 vs 认证范围

审核范围认证范围
定义一次审核活动的内容、范围、位置、单元、活动与过程、时间段认证机构颁发证书证明已覆盖的产品/服务/地理位置
4 要素内容（产品/服务）、位置、单元（部门）、活动与过程产品/服务名称、地理位置、引进项、外包过程
关系审核范围 ≥ 认证范围；审核范围是认证范围的基础

B2. 审核证据 / 审核发现 / 审核结论

术语定义来源
审核证据与审核准则有关且能被证实的记录、事实陈述或其他信息收集
审核发现将审核证据对照审核准则评价的结果（符合/不符合/改进机会）评价
审核结论审核组考虑审核目标和全部审核发现后的最终判断综合

关系链：审核证据（事实） → 审核发现（评价） → 审核结论（综合判断）—— 逐级递进。

B3. 基于风险的思维 / 过程方法

基于风险的思维

组织识别与其目标有关的风险和机遇，采取应对措施，以增强预期结果、预防不利影响、实现改进的思维方式。
应对措施考虑：① 避免风险；② 消除风险源；③ 改变可能性/后果；④ 分担风险；⑤ 接受风险。

过程方法 + 益处

将活动作为相互关联、功能衔接的过程系统进行管理，以实现预期结果。
益处：① 提高管理体系系统性；② 过程绩效可监测与改进；③ 优化资源利用；④ 增强顾客信任；⑤ 体现 PDCA 循环运行。
