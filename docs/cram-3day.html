⭐ 四大必背核心要点

① 审核组长的能力与主要职责

【能力要求】（GB/T 19011）

审核组长除具备审核员通用能力外，还应：

  1. 策划审核并有效利用资源
  2. 代表审核组与受审核方、认证机构沟通
  3. 领导审核组成员，指导见习审核员
  4. 指挥审核实施，做出审核结论判断
  5. 预防和解决审核过程中的冲突
  6. 编制和完成审核报告

【主要职责】（按阶段记）

  • 审核前：策划审核 / 组建审核组 / 制定审核计划 / 文件评审
  • 审核中：首末次会议 / 任务分配 / 进度控制 / 汇总审核发现 / 判定不符合 / 形成审核结论
  • 审核后：编制审核报告 / 跟踪验证纠正措施 / 保存审核记录
口诀：前策划 · 中指挥 · 后报告

② 基于风险的思维（ISO 9001 核心）

【定义】

组织通过识别风险与机遇、采取应对措施，以增强有效性、获得改进效果、预防不利影响。是贯穿 ISO 9001 全过程的思维方式。

【条款定位】

条款内容
0.3.3基于风险思维的说明
4.1 / 4.2识别组织环境、相关方需求中的风险
6.1应对风险与机遇的措施（策划）
9.1.3 / 9.3数据分析、管理评审中评价措施有效性
10.3改进中持续应对风险

【应对 4 步】

  1. 识别风险与机遇（内外部环境、相关方、过程）
  2. 评价风险等级（可能性 × 影响程度）
  3. 策划应对措施（规避 / 消除 / 分担 / 承担 / 利用机遇）
  4. 评价有效性并持续改进

【与"预防措施"的区别】

风险思维 = 主动、系统、贯穿全过程；预防措施 = 被动、单点。2015 版后用前者替代。

④ 案例题万能三段式模板

第一段 · 审核依据

本次审核依据 ISO 9001:2015《质量管理体系 要求》、受审核方管理体系文件、适用的法律法规（如《XX 法》《XX 条例》及行业标准），以及认证合同约定。

第二段 · 审核关注要素（4M1E + 标准章节）
  • 人（Man）：人员能力、培训、资质、意识 （7.1.2 / 7.2 / 7.3）
  • 机（Machine）：设备维护、校准、监视测量资源 （7.1.3 / 7.1.5）
  • 料（Material）：外部供方控制、进货检验 （8.4 / 8.6）
  • 法（Method）：过程控制、作业文件、运行策划 （8.1 / 8.5）
  • 环（Environment）：运行环境、基础设施 （7.1.3 / 7.1.4）
  • 测（Measurement）：监视测量、顾客满意、内审 （9.1 / 9.2）
  • 关键控制点：结合行业写 2–3 条（餐饮→食品安全；医疗→无菌；建筑→施工工艺）
第三段 · 审核方法

采用 文件审查、现场观察、人员访谈、记录查验、实物核查、数据抽样 等方法组合：

  • 文件：查管理手册、程序文件、作业指导书
  • 现场：观察关键过程运行状态与环境
  • 访谈：最高管理者（方针目标）→ 管理层（职责）→ 操作层（过程熟悉度）
  • 记录：抽样检验、培训、不合格处置、内审记录
  • 追溯：选取典型订单/项目做顺向 + 逆向追溯

💡 考场心法

  • 大题 分条写，每条前加序号 / bullet，评卷老师好给分
  • 每条先写 标准条款号 再写内容，显专业
  • 看到"结合…"类案例题，先 抄情境关键词 再作答，避免空谈
  • 时间不够时，先搭 三段式框架，再填血肉
  • 字迹工整，段落分明 —— 视觉分是真的
